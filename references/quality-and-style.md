# Quality And Style Review

Use this file for anti-AI rewriting, style extraction, originality review, final audit, and content risk control.

## Anti-AI Style Principles

AI-sounding marketing copy usually fails because it is broad, balanced, abstract, and over-neat. Make copy sound like a competent person with specific experience.

Rewrite rules:

| Weak Pattern | Better Move |
| --- | --- |
| Broad slogan | name the scene, person, decision, or object |
| Generic benefit | show the before/after or avoided mistake |
| Abstract framework | give a concrete checklist or decision rule |
| Perfectly symmetrical paragraphs | vary sentence length and rhythm |
| Overuse of "首先/其次/最后" | use causal language and scene transitions |
| Empty adjectives | replace with proof, number, action, or contrast |
| Heavy jargon | explain in buyer language first, expert term second |

Checklist before delivery:

- Does the first sentence sound like something a real creator would say on camera?
- Is there at least one concrete detail in every major paragraph?
- Can a viewer identify whether this is for them within 5 seconds?
- Is the CTA connected to the content, not pasted at the end?
- Are risky claims softened or supported?

## Style Extraction

When asked to imitate or preserve a style, extract transferable features rather than copying phrases.

Analyze:

| Layer | What To Extract |
| --- | --- |
| Persona | expert, friend, challenger, storyteller, operator |
| Rhythm | short punchy clauses, long narrative, Q&A, debate |
| Vocabulary | industry terms, local terms, emotional words, taboo words |
| Logic | case-first, conclusion-first, conflict-first, checklist-first |
| Emotion | calm, angry, amused, empathetic, skeptical |
| CTA style | soft invitation, direct consult, keyword, comment debate |

Then create a new style guide:

```text
写作人设：[persona]
语气：[tone]
句式：[rhythm]
常用表达类型：[not exact phrases]
禁止项：[overclaims, copied lines, sensitive claims]
适合内容：[topic types]
```

## Originality Review

Use a double-blind review mindset: judge whether a script stands on its own without knowing its source.

Review dimensions:

| Dimension | Questions |
| --- | --- |
| Topic independence | Is the angle distinct from common templates? |
| Evidence | Is there a specific case, process, or decision rule? |
| Language | Are there copied-sounding phrases or generic AI filler? |
| Business fit | Does it connect to the user's actual offer? |
| Audience fit | Is the scene recognizable to the target audience? |
| Platform fit | Does the hook and length fit Douyin/Xiaohongshu/Bilibili/WeChat? |

Scoring:

- `A`: publish after minor polish.
- `B`: good idea but needs specificity, proof, or clearer CTA.
- `C`: generic, risky, or too detached from the offer.
- `D`: misleading, copied, unsupported, or compliance-sensitive.

## Final Audit

Before returning marketing content, run these checks:

1. `事实`: any data, claim, law, medical/financial advice, or competitive claim needs support or caveat.
2. `承诺`: remove guaranteed outcomes, guaranteed income, guaranteed cure, or certain ranking claims.
3. `版权`: do not reproduce proprietary prompt text, third-party scripts, or recognizable creator copy.
4. `平台`: avoid bait, false urgency, sensitive events, excessive fear-mongering.
5. `商业`: ensure the CTA matches the user's real funnel and capacity.
6. `风格`: make it oral, concrete, and not mechanically formatted.

## Risk Notes By Category

| Category | Review Focus |
| --- | --- |
| Finance/investment | no guaranteed return, no personalized investment advice without qualification |
| Healthcare/beauty medical | no cure guarantees, no before/after exaggeration, include professional diagnosis boundary |
| Law/debt | no promise to eliminate debt or win cases, route to qualified professionals |
| Education/employment | no guaranteed admission, job placement, score, or salary |
| Insurance | no misleading coverage claims, explain exclusions and underwriting |
| Parenting/psychology | avoid shame or diagnosis; use supportive language |
| Luxury/jewelry/antiques | authentication and valuation need evidence and caveats |

## Review Prompt Skeleton

```text
你是营销内容终审。
请审查以下内容是否适合发布，并输出：
1. 总评分 A/B/C/D
2. 最大的3个问题
3. 具体改写建议
4. 合规和版权风险
5. 去AI味改写版
6. 发布前需要补充的证据
要求：不美化风险，不复制来源材料原文，不保留空泛套话。
```
